# Laravel Inertia React CRUD APP

A Simple CRUD Application Built using Laravel, Inertia.js, React, and MySQL. This project demonstrates modern Full-Stack Development with Seamless Frontend-Backend Integration.

## Tech Stack
- Laravel 12
- Inertia.js
- React
- MySQL
- Tailwind CSS

## Features
- Create Records
- View Records
- Update Records
- Delete Records
- Form Validation
- Responsive UI
